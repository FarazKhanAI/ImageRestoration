About Image Restoration Studio

Restoring memories, one pixel at a time

What is this application?

Image Restoration Studio is a web application that helps you restore damaged photos with precision. You can remove scratches,and blemishes from your images while preserving original details and colors.

Easy to Use

Simple three-step process from upload to download

Fast Processing

Restore images in 2-10 seconds using advanced algorithms

Color Preservation

Maintains original color consistency during restoration

How to Use the Application

1

Upload Your Image

Drag and drop or click to upload your damaged image. Supported formats: JPG, PNG, BMP, TIFF.

2

Mark Damage Areas

Use the brush tool to mark scratches you want to remove. Adjust brush size as needed.

3

Adjust Settings

Choose inpainting method (Hybrid recommended), set brush size, and adjust processing parameters.

4

Process & Download

Click "Process & Restore Image" to see the results. View before/after comparison and download your restored image.

5

Edit Image (Optional)

After processing, use the image editor to adjust brightness, contrast, saturation, and temperature.

Technical Details

Built With

Flask OpenCV Python JavaScript HTML/CSS

Processing Algorithms

  • Fast (Telea) - Quick restoration for small damage
  • Quality (Navier-Stokes) - Better for textured areas
  • Hybrid - Best overall results (recommended)

Data Handling

  • Uploaded images are processed securely
  • No permanent storage - files are deleted after processing
  • All processing happens on the server
